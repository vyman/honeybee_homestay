# Images Folder

This folder should contain the following images for the Honeybee Homestay website:

## Required Images:

### Hero Section
- **hero-bg.jpg** - Main hero background image (1920x1080px recommended)
  - Should showcase the homestay or beautiful Chikmagaluru landscape

### About Section
- **about.jpg** - Homestay exterior or scenic view (800x600px recommended)

### Rooms Section
- **deluxe-room.jpg** - Deluxe room interior (800x600px)
- **standard-room.jpg** - Standard room interior (800x600px)
- **family-room.jpg** - Family suite interior (800x600px)

### Gallery Section
- **gallery-1.jpg** - Coffee plantation view (600x600px)
- **gallery-2.jpg** - Homestay exterior (600x600px)
- **gallery-3.jpg** - Mountain/landscape view (600x600px)
- **gallery-4.jpg** - Room interior (600x600px)
- **gallery-5.jpg** - Dining area (600x600px)
- **gallery-6.jpg** - Garden/outdoor area (600x600px)

## Image Guidelines:

- Use high-quality images (JPEG format recommended)
- Optimize images for web (compress to reduce file size while maintaining quality)
- Recommended tools: TinyPNG, ImageOptim, or Squoosh
- Ensure images are well-lit and showcase the best features of your homestay
- Consider using professional photography for best results

## Placeholder Images:

If you don't have images ready, you can use placeholder services temporarily:
- https://picsum.photos/ (e.g., https://picsum.photos/800/600)
- https://placeholder.com/ (e.g., https://via.placeholder.com/800x600)

Simply replace the placeholder URLs with your actual image paths once ready.
